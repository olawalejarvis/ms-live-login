#LK NIGHTWATCH

## Local setup
- Install NodeJS
- Clone the repository
- Change your working directory into the project
- Run `npm install`
- Raname `.sample.env` to `.env` and update it's values
- Run `npm test`

## Technology
- [Nightwatch](https://nightwatchjs.org/)
- [chromedriver](https://www.npmjs.com/package/chromedriver)
